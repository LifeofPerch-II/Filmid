kasutades teinud mängu. Alguses oli pisut meelde tuletamist, kuid kõik seni on olnud tehtav
b) Aega on kulunud mõned tunnid, peamiselt API kasutamise õppimiseks ja eelnevalt mainitud 
meeldetuletamiseks
c) Plaanin saada info ja ka postri otse Tkinteri aknasse ning lisaks üritan ka leida viisi,
kuidas saada sarnaste filmide kohta infot, et saaks soovitusi teha programmi abil. Lisaks
soovin teha akna ilusamaks ja vaadata, et kui selle suurust muuta, siis jääksid proportsioonid
ikka ilusti paika.
